# YogaCyberTrainer
Yoga Cyber Trainer to program ułatwiający wykonywanie ćwiczeń.<br>
Napisany w Python oraz PySide.

Jest to projekt wykonany na rzecz przedmiotu KCK na Politecnice Łódzkiej.

## Instalowanie i uruchamianie
1. Zainstaluj [Python 3.12](https://www.python.org/downloads/latest/python3.12/)
2. Pobierz program
    - `git clone https://github.com/RomanVoitovych258502/YogaCyberTrainer.git`
    - Kliknij na zielony przycisk Code -> Download ZIP
3. Otwórz terminal w katalogu programu
4. Wykonaj `pip install -r requirements.txt`
5. Aby uruchomić program wykonaj `python3 main.py`
